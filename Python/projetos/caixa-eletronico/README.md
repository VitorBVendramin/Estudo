# 🏧 Caixa Eletrônico

Simulação de caixa eletrônico desenvolvida em Python com operações bancárias básicas, extrato com data e hora e validações de segurança.

---

## 📋 Funcionalidades

- 💸 Sacar — realiza saque com validação de saldo
- 💰 Depositar — realiza depósito com validação de valor
- 👁️ Consultar Saldo — exibe o saldo atual
- 📄 Ver Extrato — exibe histórico completo de operações com data e hora

---

## 🎯 Regras

- Não é possível sacar mais do que o saldo disponível
- Não é possível depositar valores menores ou iguais a zero
- O extrato registra cada operação com data e hora automaticamente
